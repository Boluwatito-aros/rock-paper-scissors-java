# Rock, Paper, Scissors

A fully playable Rock, Paper, Scissors game built in Java.

## About
Developed a console-based Rock, Paper, Scissors game in Java, leveraging OOP principles
including classes, methods, and encapsulation.

## Features
- Computer opponent powered by Java's random number generator
- Win/loss/draw tracking across multiple rounds
- Running score summary at the end of each session
- Input validation to prevent invalid moves from breaking the game

## Tech
Java · OOP · Random number generation · Input validation
